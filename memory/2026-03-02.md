## 📊 记忆系统架构

```
Layer 1: 工作记忆 (~2000 tokens)
  └─ snapshot.md - 每次对话自动注入

Layer 2: 长期记忆 (JSONL)
  ├─ facts.jsonl - 确定的事实
  ├─ beliefs.jsonl - 推断的信念
  └─ summaries.jsonl - 摘要

Layer 3: 文件系统 (Markdown)
  ├─ preferences/ - 用户偏好
  ├─ relationships/ - 关系网络
  ├─ knowledge/ - 知识领域
  └─ context/ - 上下文

Layer 4: 原始日志
  └─ YYYY-MM-DD.md - 每日对话记录
```
