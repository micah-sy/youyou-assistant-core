## 🚨 故障排查

### 问题 1：Obsidian 看不到新文件

**解决：**
```
1. 右键文件树 → "Reload"
2. 或关闭 Obsidian 重新打开
3. 检查文件是否在 workspace/ 目录
```

### 问题 2：Git 同步失败

**解决：**
```bash
# 检查 Git 状态
cd /home/admin/.openclaw/workspace
git status

# 检查远程仓库
git remote -v

# 重新推送
git push -u origin main
```

### 问题 3：OpenClaw 不保存记忆

**解决：**
```bash
# 检查工作区路径
ls -la /home/admin/.openclaw/workspace/memory/

# 重启 OpenClaw
openclaw gateway restart
```
